# React Node Auth System

A simple full-stack authentication system built with React, Node.js, and Express. This project demonstrates secure user registration and login using modern backend practices.

## Live Demo
https://react-auth-kohl.vercel.app/login

## Features
- User Registration
- User Login Authentication
- Password Hashing (bcrypt)
- JSON Web Token (JWT) Authentication
- Protected Routes
- Basic Error Handling

## Tech Stack
### Frontend
- React
- Tailwind css
### Backend
- Node.js
- Express.js
- MongoDB
### Authentication
- JWT (JSON web token)
- Bcrypt

## Learnings
- Implementing authentication using JWT
- Securing passwords with hashing
- Structuring a full-stack application
- Handling API requests between frontend and backend
